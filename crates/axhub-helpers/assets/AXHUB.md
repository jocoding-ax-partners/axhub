# AXHUB Data Context

This directory is generated by `axhub-helpers sync`.

- `catalog.json` is private local context and must not be committed.
- `AXHUB_TARGET` records the selected runtime target for snippets.
- Use `axhub catalog search --json --limit 200` for discovery.
- Use `axhub catalog get --connector <name> --path <path> --json` before live reads.
- Use `axhub catalog invoke --action read --execute --json` only after first live read consent.
